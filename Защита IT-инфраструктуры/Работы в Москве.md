# Настройка DHCP на MSK-RTR

Ставим пакет:

```
apt install isc-dhcp-server
```

Редактируем конфиги:

```
nano /etc/default/isc-dhcp-server
```

Проверяем строки в файле:

```
DHCPDv4_CONF=/etc/dhcp/dhcpd.conf # Основной файл конфигурации
INTERFACESv4="ens4" # Внутренний интерфейс
```

Редактируем основной конфиг:

```
nano /etc/dhcp/dhcpd.conf
```

```
subnet 192.168.1.0 netmask 255.255.255.0 {
    range 192.168.1.50 192.168.1.100;
    option routers 192.168.1.1;
    option domain-name-servers 192.168.1.2, 77.88.8.1;
    option domain-name "effort.cool";
    default-lease-time 600;
    max-lease-time 7200;
}
```

Перезапускаем сервис:

```
systemctl restart isc-dhcp-server
```
