# Настройка сетевых интерфейсов

> **Проверить названия сетевых интерфейсов на рабочем стенде, если что не забыть поменять названия. Работаем от root!!!**

## Оглавление

## ЦОД

### DC-RTR-1

Останавливаем NetworkManager:

```
systemctl stop NetworkManager && systemctl disable NetworkManager
```

Открываем /etc/network/interfaces:

```
nano /etc/network/interfaces
```

Настраиваем сетевые интерфейсы:

```
# External address

auto ens3
iface ens3 inet static
address 200.100.100.20
netmask 255.255.255.0
gateway 200.100.100.10

# Internal address

auto ens4
iface ens4 inet static
address 10.15.10.3
netmask 255.255.255.0
```

Перезапускаем networking:

```
systemctl restart networking
```
